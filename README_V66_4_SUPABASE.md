# V66.4 – Histórico e Evolução Supabase

Esta versão adiciona a leitura histórica diretamente do Supabase para a aba Evolução e para o painel institucional.

## Principais mudanças

- Evolução por turma usando dados salvos em `avaliacoes` e `resultados_alunos`.
- Comparação entre avaliações salvas na nuvem.
- Evolução por descritor a partir de `descritores_json`, `gabarito_json` e `respostas_json`.
- Evolução individual por aluno.
- Painel institucional da nuvem em Gestão com médias, variação e alunos em Elementar I/II.
- Botão **Atualizar histórico da nuvem** na aba Evolução.
- Cache atualizado para V66.4.

## Observação

Faça login no Supabase antes de abrir a aba Evolução para carregar o histórico institucional.
