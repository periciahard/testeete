# Diagnóstico Pedagógico ETE — V65.3

## Ajuste principal
Menu lateral esquerdo redesenhado em formato compacto, com foco em caber 100% na tela em desktop Full HD, sem rolagem lateral desnecessária.

## Alterações
- Largura do menu reduzida para aproximadamente 220px.
- Logo e identificação institucional compactadas.
- Botões do menu com menor altura, menor espaçamento e fonte ajustada.
- Botão Sair mantido fixo no rodapé do menu lateral.
- Área central ampliada com `margin-left` ajustado.
- Regras responsivas mantidas para telas menores e celulares.
- Cache/versionamento atualizado para V65.3.

## Observação
Em telas muito baixas, o menu passa a permitir rolagem vertical apenas como fallback, preservando o acesso a todos os itens.
